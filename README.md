<h1>Rentalx</h1>
<p>
  Esta é uma api feita com base nas aulas do Ignite de Node.js da RocketSeat,
  esta aplicação tem o objetivo de ajudar empresas de alugueis de carros no
  gerenciamento de clientes e do seu site.
</p>
